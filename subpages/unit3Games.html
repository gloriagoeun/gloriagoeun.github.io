The Shoplifter & Bananas v. Cabbage

Access to code

To showcase my understanding of Rust and game engine programming, I created two small 2D games (The Shoplifter, Bananas vs. Cabbage) using one simple engine in Rust with my classmate Josh Yum.

Our first game is titled "The Shoplifter", and the objective for the game is for the player to shoplift all of the items on the list without getting caught by the associates. Our second game "Bananas vs. Cabbage" is a multiplayer game where players compete with each other to see who can fill the screen up the most with either bananas or cabbages. Please feel free to check out our games and play them locally on your computer!

One thing we really wanted to work on with this project was creating an engine where we could reuse aspects of our first game with our second. Although there are still some repeating parts of the main files of game1 and game2, we were able to accomplish the separation of the engine from games 1 and 2, as you can see from the organization in our game repo, which is something we learned a lot from while coding the game. We were also able to learn how to incorporate sound, using the Kira Rust library.
