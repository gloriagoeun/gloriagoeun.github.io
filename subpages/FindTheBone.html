Find the Bone! Return Home!

Access to code

To showcase my understanding of Java and object-oriented programming, I created this 2D Game informally titled "Find the Bone! Return Home!" from scratch. The goal of this game is for you, as the lovely dog character, to retrieve five bones placed somewhere in the map and then return 'HOME' as fast as you can. Along the way, you may also find different items, such as the up and down arrows, that increase and decrease your speed. The game was written with the help from the tutorial/video series 'How to Make a 2D Game in Java' by RyiSnow on YouTube. All of the pixel art and music was created by me using the platforms Piskel and BeepBox.

Through this process, I was able to learn a lot more about the mechanisms behind running and designing a game, such as how to best format the game loop and tricks to best optimize and improve rendering performance. This project was a joy to develop and design, so I hope you enjoy walking around my world! Woof woof!
