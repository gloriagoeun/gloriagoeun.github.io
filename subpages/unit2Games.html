River & Space Ducky 2D Game

Access to code

To showcase my understanding of Rust and game engine programming, I created these 2D Games "Space Ducky" and "River Ducky" from scratch with my teammates Sophia Ristuben, Evelyn Hasama, and Rachel Yang. The goal of "Space Ducky" is for you, the lovely duck, to avoid colliding with the space objects and reach the top successfully. The goal for "River Ducky" is for the duck to reach the top by jumping onto logs and lilypads without falling into the river along the way.

From coding up the game, we learned more about creating sprites from the spritesheets, moving sprites, checking collisions, and rendering full screen images. We also learned about modularizing code to improve the readability and organization of our code. We could easily reuse code for the second game, as most of the logic was the same.
